Итак, правила указания правил обработки. Что.



У каждого изображения есть домен хостинга, на котором оно располагается. Именно по доменам в основном и будет происходить отбор.
Все директивы доменов проверяют указанные строки на вхождение в домен фото. Т.е. INCLUDE abc добавит все домены содержащие подстроку abc.

Для добавления домена в список обрабатываемых укажите перед ним директиву INCLUDE. Например,

INCLUDE example.com

добавит example.com в список доменов.

Для исключения домена из списка обрабатываемых укажите перед ним директиву EXCLUDE. Например,

EXCLUDE example.com

исключит example.com из списка доменов.

Для указания всех доменов используется символ * (звёздочка). Например:

INCLUDE *

добавит все домены в список обрабатываемых.

Для ограничения размера картинок в байтах используются директивы LESSTHAN и MORETHAN. Например,

MORETHAN 4096
LESSTHAN 1048576

укажет игнорирование картинок размером больше одного мегабайта и меньше четырёх килобайт.
